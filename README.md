# GLOBAL AIRBNB PERFORMANCE DASBOARD

Just Built My Airbnb Dashboard – Here’s What I Learned!

Excited to share my latest project: an interactive Airbnb Dashboard built using real-world datasets sourced from Maven Analytics, focused on turning raw data into clear and actionable insights 📊

💡 What I Learned:

• Understanding how DAX works in Power BI
• Using DAX functions like COUNT, AVERAGE, and conditional logic
• Cleaning and transforming data using Power BI (Power Query)
• Creating and using Bookmarks to improve dashboard navigation
• The importance of choosing the right color theme for better visualization
• Extracting key insights from data by analyzing charts and patterns
• How attention to detail improves dashboard quality
• Structuring pages and naming them properly for better user experience

🛠️ Tools I Used:

• Power BI (data transformation, dashboard creation & visualization)
• Power Query (for data cleaning & transformation)
• DAX (measures, calculations & logic)

📊 Data Source:

• Maven Analytics (dataset downloaded for analysis and dashboard creation)

📈 Key Takeaway:

This project helped me understand how small details like design, layout, and clear insights play a big role in making dashboards more effective and user friendly.

I’m continuously improving my data analytics skills, and feedback is always welcome!
